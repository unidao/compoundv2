Compound Protocol
=================

#### 1) Make environment
.env

#### 2) Build

```
$ cd compoundv2
$ npm install 
$ truffle compile
```
#### 3) Deploy
```
$ truffle migrate --network rinkeby
```
#### 4) Discover
list of addresses:

```
$ cd/networks..
```
list of ABI:
```
$ cd/config...
```
